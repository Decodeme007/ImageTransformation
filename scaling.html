Image Scaling

About Scaling

Image scaling is a transformation that resizes an image — either enlarging (upscaling) or reducing (downscaling) its dimensions. This operation changes the size of the image but preserves its structure and proportions when done uniformly.

Mathematical Formula

Scaling is represented by multiplying the original coordinates with scaling factors Sx and Sy: 

x' = x · Sx
y' = y · Sy

Where:

  • (x, y) → original coordinates
  • (x', y') → coordinates after scaling
  • Sx → scaling factor in X-direction
  • Sy → scaling factor in Y-direction

Python Code (Using OpenCV)


import cv2

# Read the image
image = cv2.imread('image.jpg')

# Define scaling factors
scale_x = 1.5   # Horizontal scale
scale_y = 1.5   # Vertical scale

# Perform scaling using cv2.resize()
scaled = cv2.resize(image, None, fx=scale_x, fy=scale_y, interpolation=cv2.INTER_LINEAR)

# Display result
cv2.imshow('Scaled Image', scaled)
cv2.waitKey(0)
cv2.destroyAllWindows()

Explanation

The cv2.resize() function allows you to change the image size using fx and fy as scale factors. Setting both greater than 1 enlarges the image, while values between 0 and 1 shrink it.
