Image Rotation

About Rotation

Image rotation is a geometric transformation that rotates an image by a specific angle around a given point (usually the image center). It changes the orientation of the image while preserving its shape and pixel values.

Mathematical Formula

The rotation of a point (x, y) around the origin by an angle θ (in radians) is given by: 

x' = x·cos(θ) - y·sin(θ)
y' = x·sin(θ) + y·cos(θ)

Where:

  • (x, y) → original coordinates
  • (x', y') → new coordinates after rotation
  • θ → rotation angle (positive = counter-clockwise)

Python Code (Using OpenCV)


import cv2
import numpy as np

# Read image
image = cv2.imread('image.jpg')

# Get image dimensions
(h, w) = image.shape[:2]
center = (w // 2, h // 2)

# Define rotation matrix
angle = 45  # degrees
scale = 1.0
M = cv2.getRotationMatrix2D(center, angle, scale)

# Apply rotation
rotated = cv2.warpAffine(image, M, (w, h))

# Display result
cv2.imshow('Rotated Image', rotated)
cv2.waitKey(0)
cv2.destroyAllWindows()
