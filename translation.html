Image Translation

About Translation

Image translation is a geometric transformation that shifts an image from one location to another. It moves every pixel of the image by a fixed distance in the X and Y directions without altering its shape, size, or orientation.

Mathematical Formula

Translation is defined by adding translation factors Tx and Ty to the original coordinates: 

x' = x + Tx
y' = y + Ty

Where:

  • (x, y) → original coordinates
  • (x', y') → translated coordinates
  • Tx → translation distance along X-axis
  • Ty → translation distance along Y-axis

Python Code (Using OpenCV)


import cv2
import numpy as np

# Read the image
image = cv2.imread('image.jpg')

# Define translation distances
Tx = 100   # Shift in X direction
Ty = 50    # Shift in Y direction

# Create translation matrix
M = np.float32([[1, 0, Tx], [0, 1, Ty]])

# Perform translation using warpAffine()
translated = cv2.warpAffine(image, M, (image.shape[1], image.shape[0]))

# Display result
cv2.imshow('Translated Image', translated)
cv2.waitKey(0)
cv2.destroyAllWindows()

Explanation

The cv2.warpAffine() function applies the transformation matrix M to move the image. Positive Tx values move the image right, and positive Ty values move it down.
